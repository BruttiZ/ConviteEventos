# Código de conduta

Este projeto segue o Contributor Covenant como referência prática para colaboração saudável.

## Nosso compromisso

Queremos uma comunidade aberta, respeitosa e profissional. Pessoas de qualquer nível de experiência, origem, identidade, idioma ou localização devem conseguir participar sem medo de humilhação, assédio ou exclusão.

## Comportamentos esperados

- Usar linguagem acolhedora e objetiva.
- Respeitar opiniões e experiências diferentes.
- Aceitar feedback com boa-fé.
- Focar no que é melhor para o projeto e para as pessoas que o usam.
- Explicar decisões técnicas com clareza.

## Comportamentos inaceitáveis

- Assédio, insultos, ataques pessoais ou intimidação.
- Comentários discriminatórios.
- Exposição de informações privadas sem permissão.
- Provocação repetida ou comportamento hostil.
- Pressão para aceitar mudanças sem discussão técnica.

## Aplicação

Mantenedores podem remover comentários, fechar issues, recusar contribuições ou bloquear participantes que violem este código.

Se precisar reportar um problema, abra contato privado com o mantenedor do repositório. Relatos serão tratados com cuidado e discrição.
